# MeteoNet

Simple meteorological net with single root node and multiple sensor nodes.  
Focusing on low power consumption, low data TX and near/far communication.

### FW
> Written in C++  
> No runtime memalloc strategy  
> No unneccessary C++ features

### HW
> Developed in KiCAD  
> Simple design  
> One HW, multiple roles strategy
